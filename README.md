# order-management-system
Production-grade Order Management REST API built with Java 17, Spring Boot, PostgreSQL, and Docker.


The purpose of this project is to develop a backend Order Management System that manages users, products, and customer orders through RESTful APIs. It allows administrators to maintain product inventory and enables customers to place orders while automatically validating stock availability, calculating order totals, and updating inventory. The project demonstrates how to build a scalable, maintainable, and production-ready REST API using Java, Spring Boot, PostgreSQL, and Docker.
